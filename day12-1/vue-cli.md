1. 全局安装
```
npm install vue-cli -g
```
2. vue-cli 完整版
```
vue init webpack my-project
```
```
build webpack配置文件
config 整体项目配置文件
src -》 main.js入口文件 App.vue根组件 components组件目录 js css 源码
index.html 模板文件
static 静态资源文件夹
package.json
```
3. csp 模式   轻量级的
```
vue init webpack-simple my-project
```
```
src -> main.js 入口文件 App.vue 根组件
webpack.config.js
package.json
```
